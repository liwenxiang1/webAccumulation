# 文件权限

> 查看文件夹权限信息
```
cacls node-test

C:\tools\WorkPlace\node-test BUILTIN\Administrators:(OI)(CI)F
                             NT AUTHORITY\SYSTEM:(OI)(CI)F
                             BUILTIN\Users:(OI)(CI)R
                             NT AUTHORITY\Authenticated Users:(OI)(CI)F
```

> 禁用文件夹权限
```
cacls node-test  /t /e /c /r "Authenticated Users"

处理的目录: C:\tools\WorkPlace\node-test
...


cacls node-test
C:\tools\WorkPlace\node-test BUILTIN\Administrators:(OI)(CI)F
                             NT AUTHORITY\SYSTEM:(OI)(CI)F
                             BUILTIN\Users:(OI)(CI)R
```


> 获取文件夹权限
```
cacls node-test /t /e /c /g "Authenticated Users":f

处理的目录: C:\tools\WorkPlace\node-test
...

cacls node-test

C:\tools\WorkPlace\node-test BUILTIN\Administrators:(OI)(CI)F
                             NT AUTHORITY\SYSTEM:(OI)(CI)F
                             BUILTIN\Users:(OI)(CI)R
                             NT AUTHORITY\Authenticated Users:(OI)(CI)F
```
